    靖州覃氏 覃黄团

    覃黄团

    谱书提到的覃黄团, 即覃鳌与黄龟年共筑一宅, 隐居于湖南渠阳, 名曰覃黄团.

    靖州渠阳

    太平兴国五年（980年），“土人杨通宝入贡，命为诚州刺史”, 侗族大姓杨氏, 占今靖州渠水河滩头一带;
    元丰五年（1082年），析沅州之贯堡砦置渠阳县，隶属诚州。
    元祐二年（1087年）七月改为渠阳军，三年废军州为砦，隶属沅州。
    元祐五年复置诚州及渠阳县，为羁縻州。
    崇宁二年（1103），杨晟臻纳贡归附，遂以安靖之意，改诚州为靖州（靖州称谓始于此）。
    后来朝代更迭, 靖州管过会同,通道,绥宁等临近县.
    1997年,怀化地区改为怀化市, 靖州县, 会同, 通道归怀化市, 绥宁归邵阳市; 今天的靖州苗族侗族自治县, 城关镇为渠阳镇.

    《宋史 黄龟年传》卷三百八十一

    黄龟年，字德邵，福州永福人(今福建永泰县),生于北宋元佑三年(1083年)。
    登崇宁五年(1106年)进士第，调洺州司理参军，累官河北西路提举学士; 吕颐浩见而奇之，入为太常博士; 靖康元年，除吏部员外郎，拜监察御史，寻除尚书左司员外郎、中书门下检正房公事，充修政局检讨官。乞令检正官察通进司，帝从其请; 时颐浩再相, 迁殿中侍御史, 主弹劾秦桧; 除太常少卿，累迁起居舍人、中书舍人兼给事中.
    侍御史常同(1090—1150, 宋邛州临邛今四川邛崃人)言龟年阴结大臣，致身要地，又交结诸将，趣操不正，罢归。司谏詹大方希桧意劾龟年附丽匪人，搢绅不齿，落职，本贯居住。卒，六十三(1144年)。
    龟年微时，永福簿李朝旌奇之，许妻以女。龟年既登第，而朝旌已死，家贫甚。或劝龟年别娶，龟年正色曰：“吾许以诺，死而负之，何以自立。”遂娶之。任子恩，先奏其弟之子，人皆义之。子衡，仕至湖南提举。

    《中湘黄氏族谱》《渠阳黄氏世谱》

    据《中湘黄氏族谱》记载，丰城沇江始祖黄瑕长子名黄睿。
    黄睿长子黄瞎娶妻范氏，封宜人，生五子：长子黄高；次子黄昊，徙居湖北黄州；三子黄良河，迁安徽祁门；四子黄良社，迁江西乔坑：五子黄良稷，迁江西铜坑。
    黄高一支，留居丰城沇江，光大祖宗业绩。黄高初任承仕郎，封为芒水侯。娶妻魏氏，封夫人，生二子：黄巽、黄均。黄巽、黄均兄弟二人的子孙，都功名显赫。
    黄均又名黄君章，官任柱国将军，后父以子贵，赠大中大夫。黄均生9个杰出儿子，长子黄德阳和兄弟九人，竟然一门八进士(余一人是举人)、九尚书，其中还有二人高中状元，夺得科举考试中的最高名次。连女婿陈氏官至刑部尚书，江西丰城沇江有”九子十尚书”的美称。

    黄德阳，仁宗景祐初年(1034年)中进士，官任吏部尚书。因与朝政见不合，隐居吉安府吉水县鹅劲大丘圳脚(今吉水县富滩镇江头村），为鹅劲大丘圳脚黄氏始祖。黄德阳娶妻范氏，封贤德夫人，生四子：黄映壁、黄咏壁、黄辕壁、黄棣壁。 黄映壁，字进科，北宋官任朝议大夫、左副都御史，元祐六年即公元1092年，携带小龟年在内亲眷调任南京，居南京十字街洗马巷，辞官后，隐居福建永福(今福建永泰县)。
    黄映壁娶妻谢氏，生三子：长子黄俊朝，字廷植，官任浙江提刑按察司，其二子秀文、秀武后裔南迁黔阳等地；次子黄俊国，便是著名的忠臣黄龟年，为渠阳黄氏始祖; 三子黄俊生，其子四黄良山、良和、良社、良穆分支湘扛、洗江，并门板桥等地.

    黄龟年三代迁徒三省三地，其走线是；江西丰城县沇江—江西庐陵吉水县—福建永福县—湖南靖州渠阳。黄龟年出生在庐陵吉水，成长在福建永福，定居在湖南渠阳;

    据《渠阳黄氏世谱》记载，其始祖黄龟年, 谱名黄俊国，字德邵，号鲁颈，是江夏黄香的第33代，丰城黄氏的第7代，为庐陵吉水县鹅劲大丘圳脚（今吉水县富滩镇江头村）黄氏第三代孙。早年随父转南京，迁福建永福。崇宁五年(1106年)进士第, 屡得升迁, 参见前文<<宋史>>, 与秦桧为政敌.
    绍兴六年(1138)苗乱，黄龟年率领明星、潘有月、姚北大和蒙万户四将出征，由辰州进兵至古州 [今贵州榕江县]八湾; 平定之后, 萌生退意，定居渠阳。

    绍兴十四年(1144年)，派遣谭鳌至渠阳，宣布诏封黄龟年为“兴国侯”的圣旨, 黄龟年劝前来宣诏的谭鳌也留居不归，谭怕获罪，便变姓名，去言为“覃”，与黄龟年同隐于渠阳，这就是靖州县“黄覃团”的由来。
    因黄龟年隐居渠阳而得名的地方还有, 龙宝乡上黄寨, 该地原名领金山。
    宋孝宗淳熙三年[1176年], 黄龟年去世，葬在靖州贯头乐山坡[今靖州县甘棠镇平原村], 黄龟年的墓庐，至今仍在，1985年，再次将黄龟年墓依原有规模修复，1987年，靖州县人民政府将黄龟年墓宣布为县级重点文物保护单位.
    传说当时黄龟年去世后，宋孝宗曾下令将他的遗骸水运回京国葬。一日，运柩船停泊会同县高涌，夜间忽然狂风大作，浪高水涌。待到天明，风平浪静，而船却复泊于原地。如此的现象，屡次发生。黄氏族人认为是神灵要求，便将黄龟年归葬靖州甘棠镇，而会同县的“高涌”，即因此得名。
