    靖州覃氏 湖贡宗支

    湖贡宗支

    本宗支以太湖, 太贡二公为覃氏始祖;
    湖贡二公迁入湖南住七年, 又迁入江西吉安府泰和县金鸡巷鹅江大丘.
    湖贡二公后辈又迁入湖广省长沙府靖州直隶, 以覃为姓氏.

    太湖流宗

    太才海秀 再正通光
    朝大时尚 学宗圣化 德启昭(文)明 兴邦定国 辅世安人(显荣)

    太湖公之后落业于靖州文溪(地名), 流宗六团合议字辈如上.
    秀字先祖迁入大足珠溪, 立宗祠, 张献忠之乱后于康熙三十三年(公元1694年), 再度由湖广迁入四川大足, 现族人八千, 分布于大足珠溪, 龙水等地.

    太贡流宗

    再顺天心 志政通秀
    朝启国士应 斯宗永远长 继述绍先德 恩光万世昌
    宏仁学义定 昭进保忠良 富贵荣华久 登高显达章
    伦常礼让逊 福禄振家声 廉朴为根本 益风耀乾坤

    太贡公之后落业于靖州尧馆(地名,上团太平坊四甲新分,中团尧馆老宅墓脚坪,下团泡渡大村), 流宗三团合议字辈.

    本太贡支入川(指重庆和四川, 公元1997年重庆升为直辖市)有比较明确的时间记载, 见于史籍(参考中国覃氏通书154页).
    康熙33年(公元1694年)启字辈祖入川, 先于合川北门外乘角寺.
    康熙36年(公元1697年)于资中比门外方家冲.
    康熙42年(公元1703年)迁入大足县,
    启舜公落业西盐井滩,
    启宏公落业龙水镇街坊, 干坝子,
    启昌公落业帽和山,
    启让公落业桃子湾,
    启和公落业中敖镇,
    启焕落业袁子坪,

    继后入川有,
    启灼购买余成斗之业居住,
    启享, 启篆落业湾滩桥, 河沟桥.

    太贡三驱季家启鹍流宗

    康熙59年(公元1720年)本支启鹍祖(生于康熙26年,公元1687年)携家眷(陶氏生国榜)由湖南入四川.
    乾隆13年(公元1749年)国榜公(号显楚)于石桥湾置业购买田地(此时启鲲祖62岁,见于地契), 期间辗转奔波劳作三十年.
    国榜公之后, 斯玖公后裔, 分布于大足三驱镇石桌子杀人坳, 季家镇花桥覃家大桥(桥接曾家坝和龙江, 响水滩水库边), 季家镇龙江, 清坪一带(陈泡冲, 乱草湾, 清明山). 从地图上看, 这几个地点都集中在一个区域.
